# 🍺 Homebrew Tap

Homebrew tap for [aresnasa](https://github.com/aresnasa)'s macOS tools.

## Formulae

| Cask | Description |
|------|-------------|
| `keyvalue` | K🔒V — 安全的密码与键值对管理工具 \| Secure password & key-value manager |

## Usage

```bash
brew tap aresnasa/tap
brew install --cask keyvalue
```

Or in one line:

```bash
brew install --cask aresnasa/tap/keyvalue
```

## Uninstall

```bash
brew uninstall --cask keyvalue
brew untap aresnasa/tap
```
